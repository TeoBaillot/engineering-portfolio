# Jeu de Pendue – Site Web

## Contexte
Projet d'étude visant à créer un jeu classique de pendu accessible depuis un navigateur web.  

## Objectif
Développer une application web interactive pour permettre aux utilisateurs de jouer au pendu directement dans leur navigateur.

## Fonctionnalités
- Sélection aléatoire de mots à deviner.
- Interface graphique responsive et intuitive.
- Gestion des lettres proposées, affichage du mot et du pendu.
- Détection des victoires et défaites.

## Technologies utilisées
- **HTML** : structure de la page et éléments interactifs.  
- **CSS** : mise en forme et styles visuels.  
- **JavaScript** : logique du jeu, interactions et mises à jour dynamiques.

## Utilisation
1. Cloner le dépôt ou télécharger les fichiers.  
2. Ouvrir `index.html` dans un navigateur web moderne.  
3. Jouer au jeu directement depuis la page.
